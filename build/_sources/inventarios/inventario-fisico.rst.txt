Inventario Físico
=================

El inventario físico puede hacerse de dos maneras, ingresando las compras, o
ingresando el inventario que se ha contado. En caso de que esté en proceso de
crear el inventario físico en base a lo que se ingresó al sistema, puede imprimir el
listado para la toma física, para realizar esta acción se dirige a Inventario físico y
ajustes de inventario, y luego hacer clic en la opción de Listado para toma física.

.. image:: /_static/listado_toma_fisica.png
   :alt: Listado para toma física

El sistema mostrará una ventana donde elegir cómo imprimir el listado de los
productos según sea la necesidad, puede ser por agrupación, rubro o marca, y
ordenarlos por código o por nombre. Incluir las existencias o listar sólo con
existencias.

.. image:: /_static/opciones_listado.png
   :alt: Opciones del listado

Una vez seleccionada la opción, haga clic en Imprimir.

Y se abrirá una nueva ventana donde muestra las siguientes opciones:

.. image:: /_static/opciones_impresion.png
   :alt: Opciones de impresión

Podemos elegir la vista preliminar para asegurarnos de cómo se imprimirá el listado.
Así como se muestra en la siguiente imagen.

.. image:: /_static/vista_preliminar.png
   :alt: Vista preliminar del listado
